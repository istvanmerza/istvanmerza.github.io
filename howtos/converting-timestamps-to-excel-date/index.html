Converting AD timestamps to Excel date

Feb. 23, 2023

How To convert AD / NTFS timestamps like pwdLastSet or lastLogonTimeStamp into Excel Date:

  1. Convert online: https://www.epochconverter.com/ldap
  2. Convert using Command Line: w32tm /ntte {serial}
  3. Convert using Excel formula: =(A1/10^11/8,64)-109205
